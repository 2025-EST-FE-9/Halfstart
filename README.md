# 🏀 Nike Website Clone

HTML, CSS, Tailwind CSS만을 사용하여  
Nike 공식 웹사이트의 핵심 UI/UX를 **JavaScript 없이 퍼블리싱 관점에서 재현한 클론 프로젝트**입니다.

---

## 📌 프로젝트 개요

본 프로젝트는 Nike 공식 웹사이트의 디자인과 레이아웃을 분석하여  
**순수 프론트엔드 퍼블리싱(HTML / CSS / Tailwind CSS)** 만으로 구현한 UI 클론 프로젝트입니다.

JavaScript 기반의 동적 로직 구현보다는,

- 실제 상용 웹사이트의 UI 구조 분석
- 시맨틱한 HTML 설계
- 효율적인 스타일링과 레이아웃 구현

에 집중하여 **프론트엔드 기본기와 퍼블리싱 역량 강화**를 목표로 제작했습니다.

---

## 🎯 주요 목표 및 학습 성과

- 실제 상용 웹사이트(Nike)의 레이아웃을 분석하고 구조적으로 설계하는 경험
- 시맨틱 HTML5 기반의 마크업을 통해 웹 표준과 접근성을 고려한 퍼블리싱
- Tailwind CSS 유틸리티 클래스 조합을 통한 빠르고 일관된 UI 구현
- Flexbox와 Grid를 활용한 복잡한 레이아웃 구현 능력 강화

---

## 🛠 기술 스택

- **HTML5**
  - 시맨틱 태그를 활용한 콘텐츠 구조 설계
- **CSS3**
  - Flexbox, Grid 기반 레이아웃 구성
- **Tailwind CSS**
  - 유틸리티 퍼스트 접근 방식으로 빠르고 일관된 스타일링

---

## ⚙️ 개발 환경 & 실행 방법

본 프로젝트는 **Tailwind CSS CLI 방식**으로 스타일을 빌드했습니다.  
퍼블리싱 프로젝트이지만 실제 개발 환경과 유사한 흐름을 경험하기 위해  
입력 CSS와 출력 CSS를 분리한 구조로 구성했습니다.

### Tailwind CSS 빌드 명령어

```bash
npx tailwindcss -i src/style.css -o public/output.css --watch

---

## ✨ 구현 내용 (주요 UI)

본 프로젝트는 **JavaScript 로직 없이**,  
HTML / CSS / Tailwind CSS만을 사용하여 다음 UI 요소 구현에 집중했습니다.

- **메인 페이지 전체 레이아웃**
  - Nike 브랜드 특유의 이미지 중심 구성과 시각적 계층 구조 반영
- **네비게이션 바 / 헤더**
  - 반응형을 고려한 메뉴 구조
  - 상단 고정(Fixed) 헤더 구현
- **상품 리스트 및 상세 페이지 UI**
  - Grid 기반 상품 목록 레이아웃
  - 상품 상세 정보 섹션 디자인
- **브랜드 스타일 반영**
  - Nike 공식 웹사이트의 색상, 폰트, 전반적인 분위기를 참고하여 재현

---

## 💡 프로젝트를 통해 얻은 인사이트

- **Tailwind CSS의 생산성**
  - 반복적인 커스텀 CSS 작성 없이 유틸리티 클래스 조합만으로 빠른 UI 구현 가능
- **구조 설계의 중요성**
  - 복잡한 디자인일수록 스타일링 이전에 시맨틱한 HTML 구조 설계가 프로젝트 품질과 유지보수성을 좌우함
- **UI/UX 디테일의 가치**
  - 여백, 폰트 크기, 시각적 계층 구조, 반응형 처리 등 세밀한 요소를 분석하고 구현하며 퍼블리싱 완성도 향상
